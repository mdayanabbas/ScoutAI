# ScoutAI Limitations

- No authentication yet.
- No deployed multi-user support unless separately configured.
- No automatic job applications.
- No automatic DMs.
- No LinkedIn/X integration.
- No private-platform scraping.
- External sources can be slow, noisy, incomplete, or unavailable.
- Hacker News jobs often need enrichment and manual judgment.
- Some workflow state is frontend/localStorage only.
- Matching is deterministic and conservative; it may reject roles that still deserve a human look.
- Resume extraction quality depends on file format and source content.
- Application packs and cold DMs are preparation tools; the user remains responsible for review and sending.
